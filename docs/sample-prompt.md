## ページ構成

1. **TOP** - ファーストビュー、実績数値、サービス紹介、お知らせ、CTA
2. **会社概要** - 代表メッセージ、企業理念、会社情報、沿革、アクセス
3. **サービス** - サービス一覧、各サービス詳細、施工実績
4. **採用情報** - 採用メッセージ、募集要項、福利厚生、選考フロー
5. **お問い合わせ** - フォーム、電話番号、所在地・地図
6. **プライバシーポリシー** - 個人情報保護方針
